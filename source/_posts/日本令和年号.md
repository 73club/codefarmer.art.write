2019年4月30日，日本明仁天皇正式退位，沿用31年的“平成”年号于5月1日改为“令和”

日本的程序员们疯了，银行政府企业都要更改年号，那些写死年号的系统都要更新，30年前的机器也不知道现在还有没有人敢改。

```javascript
const old_reign = '平成'
const new_reign = '令和'

if(Date.now > 1556640000) {
   system_out.replace(old_reign, new_reign);
}
```
